# 🛠️ 패치 노트 v0.0.4

**📅 배포일**: 2025/04/06

## 🔧 주요 변경 사항

### 🎮 미니게임 페이지 추가

새로운 즐길 거리를 제공하기 위해 **미니게임 페이지**를 추가하였습니다.
상단의 header에서 미니게임을 선택하여 게임을 확인할 수 있습니다.  
이제 다양한 미니게임을 [미니게임](https://maplebox.netlify.app/games)에서 만나보실 수 있습니다.  
앞으로도 정기적으로 새로운 게임들을 추가해 나갈 예정입니다.

이번 패치에 포함된 게임:
상단 header에서 미니게임을 선택하면 플레이할 게임을 선택할 수 있습니다.

- 🍎 **사과 게임**  
  숫자를 드래그하여 합이 10이 되면 사라지는 퍼즐 형식의 간단한 게임입니다.  
  → [게임 바로가기](https://maplebox.netlify.app/games/applegame)

---

### 🎯 큐브 시뮬레이션 개선

캐릭터 검색 후 아이템을 선택하여 **큐브 시뮬레이션**을 이용할 경우,  
해당 아이템에 잠재능력 또는 에디셔널 잠재능력이 설정되어 있지 않다면,  
기본 등급을 **레어(Rare)** 로 자동 지정하도록 개선하였습니다.

이제 더 직관적으로 큐브 시뮬레이션을 이용하실 수 있습니다.
