# 패치 노트 v0.0.1

**배포일**: 2025/03/29

## 새로운 기능

- **큐브 시뮬레이션** 추가
- **큐브 종류**:
  - 레드큐브
  - 블랙큐브
  - 장인의 큐브
  - 명장의 큐브
  - 에디셔널 큐브

## 사용법

### 장비 선택

1. 시뮬레이션에 사용할 **큐브 선택**
2. 무기 부위 선택
3. 옵션을 돌릴 장비의 레벨을 선택
4. **설정 적용 클릭**
5. **한 번 더 사용하기** 클릭

### 캐릭터 장비 선택

1. 검색창에 **캐릭터 검색**
2. 장비창에서 옵션을 돌릴 **아이템 선택**
3. 옵션을 돌릴 **큐브 선택**
4. **한 번 더 사용하기** 클릭
